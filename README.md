# 예정 패치노트
> 개발 예정 상황에 따라 패치 노트가 업데이트되며, 과거 업데이트 내역은 patch-notes에서 확인할 수 있습니다.

- 패치 과정에서 우선 순위의 변동 또는 추가적인 개발이 진행될 수도 있음
- 오류 사항 발생 시 해당 문제가 더 우선 순위가 높음
- 서버 패치는 한국 시간 기준 오후 11시 ~ 오후 12시에 진행

## Version 0.0
### 0.0.7
> 다국어 기반 피드 페이지 수정
- 이는 **공지사항**과 **국가 간 게시글** 구분을 선택할 수 있도록 하기 위한 조치
- 설정 국가의 언어를 따라가도록 할 예정

### 0.0.8
> 알림 서비스 일부 연계 도입
- 현재 알람은 `회원가입`과 `문의 답변`만 받을 수 있도록 되어 있음
- 댓글과 비밀번호 변경 요청 사항에 대해서도 해당 알림이 뜨도록 수정

### 0.0.9

- 현재는 시험의 기초적인 형태만 구현되어 있음
- 과거 시험 내역과 피드백을 진행할 수 있도록 추가적인 개발 진행
